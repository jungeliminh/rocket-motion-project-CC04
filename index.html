This project focuses on simulating rocket motion under the influence of gravity, fuel combustion, and air resistance. It involves calculating symbolic and numerical solutions, plotting graphs, and providing a clear, visual understanding of the rocket's trajectory.

II. MATLAB Code and Explanation

Key Features

  • Derives symbolic equations for mass, velocity, and position.
  • Simulates rocket motion using real-world physics formulas.
  • Visualizes motion with graphs: Mass vs Time, Velocity vs Time, Position vs Time.

MATLAB Code Overview

The MATLAB code calculates motion based on Newton's Second Law. The following equations govern the simulation:

  • Mass: \( m(t) = m_0 - \frac{dm}{dt} \cdot t \)
  • Velocity: \( v(t) = v_{\text{gas}} \cdot \ln\left(\frac{m_0}{m}\right) - g \cdot t \)
  • Position: Integration of \( v(t) \) to find \( x(t) \).

The program visualizes these equations numerically and symbolically for better analysis.
